## 📖 大一下学期

### 一、专业基础课

#### 📗 电路辅助设计与仿真

| 项目 | 内容 |
|:---:|:---:|
| **学分 / 学时** | 1 学分 / 16 学时 |
| **授课老师** | 俎云霄 老师 |
| **点名情况** | 云课堂扫码 |

> **📋 课程内容：** 本课程讲授使用 MATLAB 进行计算机辅助分析和仿真的基本方法，主要内容包括：MATLAB 简介，变量及赋值，矩阵的初等运算，MATLAB 的数据分析函数库，矩阵分解与变换，多项式函数库，判断及流程控制，基本绘图方法，函数功能和数值积分函数库，字符串函数库，MATLAB 程序设计及在电阻电路、动态电路和正弦稳态电路分析中的应用。

> **📝 考核内容：** 平时作业包括在 MOOC 平台完成课程观看、练习和讨论区互动；实验需到实验室完成，现场出题、现场完成、现场验收，以第一次验收成绩为准；期末考试内容出自课件。

| 考核项 | 占比 | 说明 |
|:---:|:---:|:---|
| 平时成绩 | 10% | MOOC |
| 实验成绩 | 40% | 4 次上机实验 |
| 期末考试 | 50% | 闭卷考试 |

> **🎯 参考给分：`89`**
